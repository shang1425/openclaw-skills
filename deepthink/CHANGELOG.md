# deepthink 更新日志

## v5.0 — 2026-03-27

### 核心功能

**多 Agent 协作推理**
- 把复杂问题拆解为多个子问题
- 分配给不同角色的 Agent 独立推理
- 角色：🧠分析师 / 😈魔鬼代言人 / 🎯实用主义者 / 🔮远见者
- 汇总各方观点，检测冲突，给出综合结论

**新增文件：**
- `multi_agent.py` — 多 Agent 协作核心逻辑

---

## v4.1 — 2026-03-26

**智能追问**
- 自动检测推理中缺失的关键信息
- 按优先级（阻塞 > 高 > 中 > 低）生成追问
- 等用户回答后更新推理

**统一入口**
- `__init__.py` 整合所有模块
- `DeepThinkEngine` 类统一管理
- `deepthink()` 快速函数入口
